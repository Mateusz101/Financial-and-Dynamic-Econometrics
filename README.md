##  English Version

This repository contains a collection of projects completed for the **Dynamic Econometrics** course. Below is a brief summary of each study conducted.

### List of Projects:

* **Project 1:** Analysis of day-of-the-week anomalies in returns using statistical tests of means and variances for each day.
* **Project 2A:** Selection of ARMA model parameters in Python and forecasting based on real data using information criteria.
* **Project 2B:** ARMA modeling in Python for each day of the week, including diagnostic checking.
* **Project 3A:** Preparation of an Excel spreadsheet for estimating ARMA(2,0) model parameters using the conditional maximum likelihood method.
* **Project 3B:** Derivation of expected value, variance, and autocorrelation formulas for the ARMA(2,0) model, along with calculating a specific probability.
* **Project 4A:** Estimation of ARMA(2,0) model parameters assuming a Student's t-distribution for random errors, and calculation of forecasts.
* **Project 4B:** Estimation of forecast error variance, determination of confidence intervals, and calculation of probabilities for conditional prediction distributions.
* **Project 5A:** Investigation of the ADF test power for an AR(1) process, comparing three test versions (no constant, with constant, with trend).
* **Project 5B:** Comparison of ADF test power across different sample sizes (N=20, 100, 500) and AR(1) process parameters.
* **Project 6A:** Modeling WIG20 volatility using GARCH, EGARCH, and GJR-GARCH models with three error distributions and forecasting.
* **Project 6B:** Evaluation of the forecasting ability of GARCH models for WIG20 using rolling windows for different periods.
* **Project 7A:** Examining the impact of residual autocorrelation on the Granger causality test in a VAR(1) model and comparing HAC estimators.
* **Project 7B:** Analysis of Granger test size distortions in a VAR(1) model with error autocorrelation for different sample sizes and HAC estimators.
* **Project 8A:** Estimation of a 3-equation VAR model for energy commodities with analysis of causality, IRF functions, and variance decomposition.
* **Project 8B:** Investigation of energy commodity price cointegration and estimation of a VECM model with short- and long-term dependency analysis.
* **Project 9A:** Analysis of the impact of conditional distributions in the DCC-GARCH model on correlation estimates between energy commodities.
* **Project 9B:** Comparison of various conditional variance models (GARCH, EGARCH, GJR-GARCH) within DCC-GARCH for dynamic commodity correlations.
* **Project 10A:** Construction of a minimum variance portfolio and its comparison with individual assets.
* **Project 10B:** Extension of Part A to calculate asset weights based on multivariate models.
* **Project 11A:** Granger causality analysis across sub-periods and the entire rolling window.
* **Project 11B:** Analysis of spillover effects using the BEKK-GARCH model.
* **Project 12:** Analysis of spillover effects based on the Diebold-Yilmaz matrix.
